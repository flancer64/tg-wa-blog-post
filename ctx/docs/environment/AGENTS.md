# Контекст среды исполнения (`./ctx/docs/environment/`)

- Path: `./ctx/docs/environment/AGENTS.md`
- Version: `20260212`

## Назначение

Каталог `ctx/docs/environment/` описывает эксплуатационную среду системы: runtime-параметры, внешние зависимости, инфраструктурные ожидания и инварианты окружения.

## Карта уровня

- `AGENTS.md` — текущий документ уровня `environment/`, задающий локальные границы и требования.
- `overview.md` — декларативное описание среды исполнения, зависимостей и обязательных параметров окружения.

## Границы уровня

Уровень `environment/` ограничен характеристиками окружения исполнения и не включает продуктовые смыслы, архитектурные структуры и организационные регламенты ветви `ctx/agent/`.

## Требования

1. Документы уровня фиксируют свойства среды декларативно и не подменяются инструкциями оперативной эксплуатации.
2. При изменении состава каталога `ctx/docs/environment/` раздел «Карта уровня» обновляется синхронно.
3. Текст внутри абзацев оформляется без ручных переносов строк.

## Итог

`ctx/docs/environment/AGENTS.md` определяет локальные границы описания среды исполнения и навигацию по документам уровня.
