# Эксплуатационная среда исполнения

Path: `./ctx/docs/environment/overview.md`

## Назначение уровня

Документ фиксирует инварианты среды исполнения Telegram Translation Publisher.

## Среда исполнения

Приложение исполняется в POSIX-совместимом окружении с Node.js LTS 20.x и ручным запуском CLI из каталога проекта.

## Конфигурационное окружение

Обязательные runtime-параметры задаются переменными окружения; при отсутствии обязательного параметра запуск завершается с ошибкой.

## Файловые инварианты среды

Среда предоставляет доступ на запись в `./var/data/` относительно `projectRoot`, вычисленного в bootstrap; путь хранения фиксирован и не переопределяется конфигурацией приложения.

## Сетевые предпосылки

Среда должна обеспечивать доступ к Telegram Bot API и LLM API.

## Эксплуатационные границы

Persistent-логирование и управление архивом агрегатов относятся к внешней операционной инфраструктуре и не являются обязанностью приложения.

## Безопасность окружения

Секреты окружения не публикуются в логах; доступ к `.env` при его наличии ограничен на уровне ОС.

## Итог

Уровень среды фиксирует минимально необходимые эксплуатационные предпосылки, не подменяя архитектурные, композиционные и кодовые контракты.
