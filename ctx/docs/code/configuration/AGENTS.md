# Контекст конфигурации кода (`./ctx/docs/code/configuration/`)

- Path: `./ctx/docs/code/configuration/AGENTS.md`
- Version: `20260212`

## Назначение

Каталог `ctx/docs/code/configuration/` фиксирует локальные инженерные инварианты структуры конфигурации приложения и правила её представления в коде. Уровень уточняет раздел `ctx/docs/code/` и не подменяет архитектурные или продуктовые определения.

## Карта уровня

- `AGENTS.md` — текущий документ уровня `configuration/`, задающий локальные границы и требования.
- `structure.md` — модель структуры конфигурации, состав секций и обязательные свойства параметров.

## Границы уровня

Уровень `configuration/` ограничен описанием формы и инвариантов конфигурационных данных в коде. Документы не включают инструкции по операционному использованию агента и не дублируют общие инженерные правила из `ctx/docs/code/AGENTS.md`.

## Требования

1. Изменения документов уровня сохраняют декларативный стиль и не превращаются в пошаговые процедуры.
2. При изменении состава каталога `ctx/docs/code/configuration/` раздел «Карта уровня» обновляется синхронно.
3. Текст внутри абзацев оформляется без ручных переносов строк.

## Итог

`ctx/docs/code/configuration/AGENTS.md` фиксирует локальный контекст конфигурации и навигацию по документам уровня.
