# CLI Contract

Path: `./ctx/docs/code/cli.md`

## Назначение

Документ фиксирует инженерный контракт CLI-интерфейса Telegram Translation Publisher.

## Модель интерфейса

CLI предоставляет единственный запуск run-cycle без командного дерева, диспетчеризации и альтернативных режимов.

## Аргументы

Bootstrap передаёт в `App.run()` массив `process.argv.slice(2)` без разбора аргументов. Поддержка функциональных CLI-флагов не является частью текущего контракта.

## Поведение запуска

Один CLI-запуск соответствует одному run-cycle и завершается возвратом `exitCode`, определённого `App.run()`.

## Связь с логированием

CLI не управляет уровнями логирования через аргументы.

## Итог

CLI-контракт закрепляет минимальный однорежимный интерфейс запуска и исключает расширение модели команд в рамках текущего продукта.
