# Контекст хранения кода (`./ctx/docs/code/storage/`)

- Path: `./ctx/docs/code/storage/AGENTS.md`
- Version: `20260212`

## Назначение

Каталог `ctx/docs/code/storage/` фиксирует локальные инженерные нормы слоя хранения в реализации: границы ответственности, инварианты работы с файловым репозиторием и требования к устойчивости записи.

## Карта уровня

- `AGENTS.md` — текущий документ уровня `storage/`, задающий локальные границы и требования.
- `overview.md` — обзор модели слоя хранения, его обязанностей и инвариантов реализации.

## Границы уровня

Уровень `storage/` ограничен инженерными правилами реализации хранения и не дублирует архитектурную модель из `ctx/docs/architecture/storage-model.md`, продуктовые смыслы и операционные инструкции агента.

## Требования

1. Документы уровня сохраняют декларативный стиль и фиксируют только инварианты кодового слоя хранения.
2. При изменении состава каталога `ctx/docs/code/storage/` раздел «Карта уровня» обновляется синхронно.
3. Текст внутри абзацев оформляется без ручных переносов строк.

## Итог

`ctx/docs/code/storage/AGENTS.md` определяет локальный контекст инженерного слоя хранения и навигацию по его документации.
