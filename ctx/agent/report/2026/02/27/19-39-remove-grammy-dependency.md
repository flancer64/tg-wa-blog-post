# Итерация: удаление неиспользуемой зависимости `grammy`

## Резюме изменений

Проверено использование библиотеки `grammy` в продукте: в исходном коде и тестах вхождения отсутствуют. Зависимость удалена из `package.json` и `package-lock.json`.

## Детали работ

- Выполнен поиск по проекту (`rg -n "grammy" -S .`): найдены только записи в файлах зависимостей и архивном отчёте.
- Выполнена команда `npm uninstall grammy` для корректного удаления пакета и синхронного обновления lock-файла.
- Повторно проверены вхождения `grammy`: в продуктовых артефактах библиотека не используется.
- Выполнены unit-тесты (`npm run test:unit`) после изменения зависимостей.

## Результаты

- Обновлён файл зависимостей: `package.json`.
- Обновлён lock-файл: `package-lock.json`.
- Unit-тесты пройдены: 12/12.
