# Создание AGENTS.md для отдельной ветки документации проекта

Проверь в когнитивном контексте ветку документации `PATH`. Нужно создать или обновить файлы `AGENTS.md` во всех подкаталогах этой ветки.

## Дополнительные требования

- Предварительно проанализируй все файлы `AGENTS.md` вверх по иерархии до самого корня проекта, чтобы соответствовать требованиям, в них изложенным.
- Файлы `AGENTS.md` должны соответствовать всем требованиям в совокупности (быть краткими входными инструкциями для агентов на данном уровне документации или кода).
