# tg-wa-blog-post

Сервис для публикации постов блога в Telegram через бота. Приложение читает исходный `ru`-пост, переводит его и отправляет результаты в целевые каналы.

## Подготовка

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Создайте файл `.env` на основе `.env.example` и задайте `BOT_TOKEN`.

## Запуск

### Основной режим

Основной режим читает последнее сообщение из исходного `ru`-канала в Telegram, переводит его и публикует версии для `en` и `es`.

```bash
npm start
```

Или напрямую:

```bash
node ./bin/bootstrap.mjs
```

### Ручной режим из файла

Ручной режим нужен, когда пост не был обработан автоматически и его нужно запустить повторно без чтения из Telegram.

Входной файл должен содержать только plain text одного `ru`-поста и не должен быть пустым.

```bash
node ./bin/bootstrap.mjs --source-file ./var/manual/post.txt
```

В этом режиме приложение:

- читает текст из указанного файла;
- переводит его для `en` и `es`;
- публикует переводы в целевые каналы;
- сохраняет агрегат с отрицательным `ru_message_id` формата `-YYYYMMDDHHMM`.
